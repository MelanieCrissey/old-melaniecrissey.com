## Get Dependencies
```bash		
bundle install
```

## Update Dependencies
```bash
bundle update
```

## Run the server
```bash
rake server
```

## View your site
```bash
http://localhost:4567
```

## Ready to deploy?
```bash
middleman build
git add --all
git commit -m "your commit comment"
git push heroku master
```

## Create New Article
```bash
middleman article {your_title}
```
